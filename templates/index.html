Instructions
1. Prepare your CSV Your file must include these required headers (additional columns are allowed):
AttendeeName, Street, City, PostalCode, E-mail, AttendeePhone, DateOfBirth
2. Upload & Select Upload your CSV file and select the correct Lifesaving Society form from the dropdown menu.
3. Generate & Download Click "Generate PDFs". A ZIP file containing all filled test sheets will download automatically.
Privacy Notice This tool runs entirely in ephemeral memory. No CSV files or personal data are saved to a database or disk storage.
